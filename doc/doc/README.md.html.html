<div class="pilwrap" id="react-boilerplate">

====================
React Boilerplate
  </h1>
</div>
<p>A minimal and light dev environment for ReactJS.</p>
<div class="pilwrap" id="usage">
  <h3>
    <a href="#usage" name="usage" class="pilcrow"></a>
Usage
  </h3>
</div>
<p>Clone the boilerplate and create your own git repo.</p>
<pre><code>git clone git@github.com:nolotz/react-simple-boilerplate.git
</code></pre>
<p>Install the dependencies and start the server.</p>
<pre><code>npm/yarn install
npm/yarn start
open http://localhost:3000
</code></pre>
<div class="pilwrap" id="static-files">
  <h3>
    <a href="#static-files" name="static-files" class="pilcrow"></a>
Static Files
  </h3>
</div>
<p>You can store static files like images, fonts, etc in the <code>build</code> folder.</p>
<p>For example, if you copy a file called my_image.png into the build folder you can access it using <code>http://localhost:3000/build/my_image.png</code>.</p>
<div class="pilwrap" id="linting">
  <h3>
    <a href="#linting" name="linting" class="pilcrow"></a>
Linting
  </h3>
</div>
<p>This boilerplate project includes React ESLint configuration.</p>
<pre><code>npm run lint
</code></pre>
